示例名称： 简单的示例
示例描述：
基本全部使用默认的设置。
默认设置中所有“指标”使用相同的Y轴，即各Y轴的最小值(0)、最大值(200)皆一致。
示例效果：
示例代码： 
var paper = Raphael('holder', 515, 445), values = [ 80, 150, 60, 90, 50, 122, 31, 170 ], options = {
	centerX : 255, centerY : 220,
	members : [ 'USA', 'Canada', 'Sweden', 'Ukraine', 'Uzbekistan', 'Russia', 'Belarus', 'Germany' ]
};
paper.smartRadar(options, values);
